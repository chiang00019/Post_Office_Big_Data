# I郵箱模擬與最佳化策略

## **專案簡介**
本專案旨在透過 **數據分析與模擬技術**，最佳化 **i郵箱的儲格配置**，提升利用率並降低閒置率。我們運用了 **大數據分析** 和 **蒙地卡羅模擬 (Monte Carlo Simulation)** 來評估不同儲格配置的影響，並提供決策支援。

## **專案檔案**
| 檔案名稱 | 描述 |
|----------|------|
| `dataclean_and_simulation.ipynb` | 進行數據清理、篩選、利用率計算與模擬分析 |
| `prob_distribution.ipynb` | 計算 **i郵箱使用機率分佈**，評估需求與儲格使用模式 |
| `大數據比賽_專案文件.pdf` | 研究報告，詳細說明分析方法、結果 |
| `郵箱模擬與最佳化策略提案簡報.pdf` | 競賽簡報 |

## **數據分析步驟**
1. **數據清理與標記** 
   - 篩選 **指定郵局** 的數據 (指南郵局 100171)
   - 標記 **寄件 (1) / 取件 (0)**，建立時間區段分析
   - 定義 **儲格尺寸** (小方格 / 中格 / 大格)
2. **利用率與閒置率分析** 
   - 分析 **不同時間段的使用模式** (凌晨 / 早上 / 下午 / 晚上)
   - 計算 **使用時間 (1 → 0) 與 閒置時間 (0 → 1)**
3. **蒙地卡羅模擬** 
   - 依據歷史數據，隨機生成 **每日需求數量與使用時機**
   - **計算機會成本**，找出最佳儲格配置策略
4. **結果視覺化** 
   - **直方圖**：不同大小儲格的使用時間分佈
   - **熱力圖**：各儲格在一天內的使用頻率
   - **決策曲線**：儲格數量 vs 機會成本變化

## **關鍵發現**
- **小型儲格需求較大**，但部分未被充分利用
- **中型儲格需求最穩定**，建議維持數量
- **大型儲格需求最低**，可適度減少配置
- **高峰期 (早上 9:00 & 下午 18:00) 需求較高**，建議調整配送時間
- **超過一定數量的儲格後，增加儲格對需求滿足率的影響趨於遞減**
